# Веб-приложение для Заметок

Выполнены все основные пункты ТЗ:
- Создание заметок и их отображение в виде карточек
- Возможность редактирования (при клике на заметку) и удаления заметок
- Возможность вставить картинку
- При первом запуске имеется стартовая заметка
- Сохранение заметок в localStorage

В интернете искал способ реализации выделения текста, нашел ReactQuill: https://quilljs.com/
Работает только с 18 версией реакта, либо я в чем-то не разобрался.
Также с ее помощью реализовал выбор шрифтов (далось довольно проблематично, но работает).
Шрифтов подключено немного, можно дополнить, mirza работает только с англ. текстом.

Возможны небольшие баги со стилями при изменении шрифтов/выделения текста.

# Запуск 
- cd notes-task
- npm install
- npm run dev
